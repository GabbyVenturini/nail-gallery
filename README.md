# Nail Gallery — Frontend React + Vite + Tailwind

Projeto refeito com base no visual enviado e nos requisitos do trabalho.

## Como rodar

```bash
npm install
npm run dev
```

## Login demo

### Cliente
Você pode criar uma conta pela própria tela de cadastro.

## Observação
Este MVP usa `localStorage` para simular:
- autenticação
- carrinho
- pedidos
- catálogo
- painel administrativo

Isso permite demonstrar todo o fluxo do site sem backend.
